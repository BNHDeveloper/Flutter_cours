- Cours 02 - Famous widget
- Cours 03 - [row + column widget]

## How to Use
Each folder contains a complete Flutter project for that lesson.

## Prerequisites
- Flutter SDK installed
- Dart knowledge
- IDE (VS Code or Android Studio)
